# Security Policy

## Supported Versions

Stoa is pre-v0.1. No stable release has been made yet. All reported vulnerabilities will be addressed on a best-effort basis.

## Reporting a Vulnerability

Please use [GitHub's private vulnerability reporting](https://github.com/marcoskichel/stoa/security/advisories/new) to report security issues. Do not open a public issue.

You can expect an acknowledgement within 72 hours and a status update within 7 days.
